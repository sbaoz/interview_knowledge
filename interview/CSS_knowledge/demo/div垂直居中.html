flex布局实现（元素宽度已知）
position（元素宽度已知） 距上50%，据左50%，然后减去元素自身宽度的一半距离就可以实现
position transform （元素未知宽度） 将上面例子中的 margin: -50px 0 0 -50px;替换为：transform: translate(-50%,-50%);
position（元素已知宽度）（left，right，top，bottom为0，maigin：auto ） 子元素不设置宽度和高度，将会铺满整个父级
text-align : center 给元素的父级加，可以使文本或者行级元素(例如：图片)水平居中
margin: 0 auto 使用条件：父级元素宽度可有可无，子级元素必须使块元素，而且要有宽度（否则继承父级）
line-height : 值为元素的高度，可以使元素的文本内容垂直居中
