# ref

### ref对象创建  
- 什么是ref对象  
用createRef或者useRef创建出来的对象  
```javascript
{
    current: null // current指向ref对象获取的实际内容 可以是dom元素 组件实例 或者其他
}
```
- 类组件React.createRef  
> react/src/ReactCreateRef.js
```javascript
export function createRef() {
  const refObject = {
    current: null,
  }
  return refObject;
}
```
不要在函数组件中使用createRef 否则会造成ref对象内容丢失等情况
